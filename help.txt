Voici les différentes commandes possibles : 

cwd /CHEMIN_DU_NOUVEAU_REPERTOIRE      Changer de répertoire de travail
cwd ..                                 Repertoire parent 
dele MON_FICHIER                       Supprimer le fichier nommé du réperetoire courant
list /NOM_DU_REPERTOIRE                Lister le contenu du repertoire 
list                                   Lister le contenu du repertoire courant 
mkd /CHEMIN_DU_NOUVEAU_REPERTOIRE/NOM  Créer un nouveau répertoire
rmd MON_DOSSIER                        Supprimer le répertoire
rnfr NEW_NOM ANCIEN_NOM                Renomme un répertoire/fichier 
stor NOM_FICHIER                       Copie le fichier (client) dans le répertoire client (serveur)
pwd                                    Affiche la position dans l'arborescence
